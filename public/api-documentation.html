SayItAI API

Welcome to the SayItAI API

All routes are available and need authentication except for /api/login and /api/sign-up

Please signup if you don't have an account yet or login if you already have an account to receive your token

The token is used to authenticate all routes

If you have problems with the API, please contact us at ffast2000@gmail.com

Available Routes

  • /api/login
  • /api/sign-up
  • /api/isolate-speakers
  • /api/combine-audio
  • /api/ytdl

Route Parameters

/api/login

  • email: string
  • password: string

/api/sign-up

  • name: string
  • email: string
  • password: string

/api/isolate-speakers

  • transcription: string
  • audioFilePath: string | audioFile

/api/combine-audio

  • audioUrls: array of string | audioFiles
  • transcription: string[]
  • originalAudioUrl: string

/api/ytdl

  • url: string
